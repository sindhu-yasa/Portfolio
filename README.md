# Sindhu Yasa Portfolio

Static portfolio website built with HTML, CSS, and JavaScript.

## Local Preview

Open `index.html` directly in a browser.

## Deploy

Recommended free option: GitHub Pages.

1. Push this repository to GitHub.
2. Open repository `Settings > Pages`.
3. Set the source to the `main` branch and root folder.
4. Visit the published GitHub Pages URL after the deploy finishes.

Other free options that work well for this static site: Netlify and Vercel.
